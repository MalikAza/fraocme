# Day 03 - Numeric Utility Functions

## Overview
Demonstrates numeric manipulation functions from `fraocme.math`.

## Functions Demonstrated

### Basic Operations
- `sign()` - Get -1, 0, or 1
- `digits()` - Extract digit list
- `from_digits()` - Combine digits
- `wrap()` - Modulo wrapping

### Number Theory
- `divisors()` - All divisors
- `gcd()` - Greatest common divisor
- `lcm()` - Least common multiple

## Run
```bash
fraocme run 3 --debug
```

## Key Takeaways
- `sign()` is great for movement toward targets
- `digits()` and `from_digits()` work together
- `wrap()` handles circular/toroidal grids
- `gcd()` simplifies fractions
- `lcm()` finds cycle alignments
- `divisors()` can detect prime numbers
